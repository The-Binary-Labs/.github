# .github

This repository contains the shared community health files for the [The Binary Labs](https://github.com/The-Binary-Labs) GitHub organization.

These files act as the default community standard for repositories across the organization, including:

- issue templates
- a pull request template
- contributing guidance
- a code of conduct
- a security policy
- the public organization profile

The goal is simple: make it easy for people to report bugs, suggest features, improve documentation, and contribute responsibly across all projects.

## Public issues

Public issues are welcome for:

- bug reports
- feature requests
- documentation improvements
- general project feedback

Security vulnerabilities should not be reported in public issues. Those should be reported privately according to the policy in [`.github/SECURITY.md`](./.github/SECURITY.md).
